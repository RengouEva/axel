  Rôle : Limiteur de débit en mémoire
  Utilisation : Auth : 10 requêtes / 15 min par IP
                API : 60 requêtes / minute par IP
                Headers X-RateLimit-Remaining / Retry-After

Security Headers (dans next.config.ts)
  Rôle : En-têtes HTTP de sécurité
  Utilisation :
    - Strict-Transport-Security (HSTS) : forcer HTTPS
    - X-Content-Type-Options: nosniff : empêcher le sniffing MIME
    - X-Frame-Options: SAMEORIGIN : protection clickjacking
    - X-XSS-Protection: 1; mode=block : protection XSS legacy
    - Content-Security-Policy : politique de contenu stricte
    - Referrer-Policy : contrôle du referer
    - Permissions-Policy : désactiver caméra/mic/géoloc/payment
